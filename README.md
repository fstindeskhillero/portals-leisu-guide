# portals-leisu-guide

## 项目简介

本仓库用于归档和发布一系列独立的 HTML 页面。这些页面以静态资源形式存放，不针对任何特定域名或网站，主要作为通用 HTML 页面集合进行维护和展示。

仓库内容可能涵盖文档、工具、演示页面或其他前端资源，每个页面均为独立完整文件，便于直接访问或嵌入使用。

## 目录结构

```
portals-leisu-guide/
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放主要 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

说明：

- `pages/` 目录下存放各个独立 HTML 页面，每个页面可自包含样式和脚本，或引用 `assets/` 中的公共资源。
- `assets/` 用于存储页面共用的静态文件，降低重复加载。
- 根目录下的 `index.html` 可作为导航页或入口文件（如有需要）。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务，可直接在浏览器中打开运行。
- 页面内容可能涉及演示、说明、工具或参考信息，具体用途请查看对应页面内的说明。
- 归档的页面不保证与任何特定域名或网站关联，仅作为独立资源存在。
- 每个页面应尽量保持自解释性，必要时在页面内注明版本或更新日期。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面时请遵循目录结构约定。
- 若需提交新页面，建议放置于 `pages/` 目录下，并确保文件命名清晰、无特殊字符。
- 公共资源如有更新，请同步修改引用路径，避免页面失效。
- 仓库不承诺提供持续维护或技术支持，但欢迎通过 Issues 提出建议或报告问题。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 下的任意 HTML 文件，或通过本地服务器（如 `live-server`）运行。
3. 如需部署到线上，可将整个仓库内容上传至任意静态托管服务。
